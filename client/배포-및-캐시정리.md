# 배포 절차 & 3000 포트 종료 / 휴대폰 캐시 정리

## 1. 서버에서 3000 포트 프로세스 종료 방법

### 방법 A: 창 닫기 (가장 간단)
- **START-SERVER.bat**을 더블클릭해서 띄운 **검은 CMD(명령 프롬프트) 창**이 있다면, 그 창을 **그냥 닫기**하면 3000 포트를 쓰던 프로세스가 함께 종료됩니다.
- 창 닫은 뒤에 다시 배포하고 START-SERVER.bat을 새로 실행하면 됩니다.

### 방법 B: PowerShell에서 강제 종료
1. **PowerShell**을 **관리자 권한**으로 실행합니다.
2. 아래 명령을 **한 줄씩** 실행합니다.

```powershell
# 3000 포트 사용 중인 프로세스 PID 찾기
netstat -ano | findstr :3000
```
- `LISTENING` 줄 맨 오른쪽에 나오는 숫자(예: 12345)가 **PID**입니다.

3. 해당 PID로 프로세스 종료 (예: PID가 12345인 경우):

```powershell
taskkill /PID 12345 /F
```

### 방법 C: 한 번에 종료 (PowerShell)
```powershell
Get-NetTCPConnection -LocalPort 3000 -ErrorAction SilentlyContinue | ForEach-Object { Stop-Process -Id $_.OwningProcess -Force -ErrorAction SilentlyContinue }
```

## 4. 휴대폰에서 캐시 삭제 / 시크릿(프라이빗)으로 접속

### 4-1. 휴대폰에서 캐시 삭제 방법

#### Android (크롬)
1. **Chrome** 실행 → 우측 상단 **⋮ (메뉴)** 탭
2. **설정** → **개인정보 및 보안** → **인터넷 사용 기록 삭제**
3. **캐시된 이미지 및 파일** 체크 (필요하면 **쿠키 및 사이트 데이터**도 체크)
4. **기간**: "전체 기간" 또는 "최근 1시간" 선택 후 **데이터 삭제**

#### Android (삼성 인터넷)
1. **삼성 인터넷** → **⋮** → **설정** → **개인정보 및 보안**
2. **개인 정보 삭제** → **캐시된 이미지 및 파일** 등 선택 후 삭제

#### iPhone (Safari)
1. **설정** 앱 → **Safari** → **방문 기록 및 웹 사이트 데이터 지우기**
2. 또는 Safari에서 **aA** → **웹 사이트 설정** 등에서 해당 사이트만 데이터 삭제

### 4-2. 시크릿/프라이빗으로 접속이란? (캐시 없이 접속)

- **시크릿(크롬)** / **프라이빗(사파리)** 모드 = **캐시·쿠키를 쓰지 않는 새 탭**으로 접속하는 것입니다.
- 이렇게 접속하면 **예전에 저장된 캐시를 쓰지 않아서**, 방금 배포한 최신 화면을 볼 수 있습니다.

#### Android (크롬) – 시크릿 탭
1. Chrome 실행
2. 우측 상단 **⋮** → **새 시크릿 탭**
3. 주소창에 **pda.dongasteel.co.kr:3000** 입력 후 접속

#### iPhone (Safari) – 프라이빗 모드
1. Safari 실행
2. 우측 하단 **탭 버튼** → 왼쪽 아래 **개인** 또는 **프라이빗** 선택
3. **+** 로 새 탭 열고 주소창에 **pda.dongasteel.co.kr:3000** 입력 후 접속

시크릿/프라이빗으로 접속하면 **캐시 삭제를 안 해도** 그 탭에서는 새로 받은 파일로 보이므로, "적용이 안 된 것 같다" 할 때 먼저 이 방법으로 확인해 보시면 됩니다.
